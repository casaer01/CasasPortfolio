Robo-Friends

Robo-Friends allows the user to search up the database by name to see what robot shows up. The website is responsive and flexible on any device either let it be computer or handheld device. React.js is used to create the frontend and backend of the website. Npm is used to run commands to get the website running on github. For the database, I used an api called JSONPlaceholder that provided fake user information to create cards with infomation related to the user. Profile pictures were generated with an api called "robohash" which took the name from the data from JSONPlaceholder.

Technologies used: React.js, NPM, JSON, APIs, Github
